    简介

    从 SPSS 因子分析结果中创建符合 APA 格式的表格可能比较繁琐。本教程旨在介绍一些技巧和注意事项。我们将使用 SPSS 因子分析 - 中级教程 的结果。

    简易创建 APA 格式表格

    对于各种分析，从 SPSS 输出创建 APA 格式表格最简单的方法通常是：

    1. 在 SPSS 中调整你的分析，使输出结果尽可能接近你想要的结果。更改表格布局（哪些变量/统计量进入哪些行/列？）最好也在这里完成。
    2. 将一个或多个表格复制粘贴到 Excel 或 Googlesheets 中。这是设置小数位数、字体、对齐方式、边框等的最简单方法。
    3. 将你的表格从 Excel 复制粘贴到 Word 中。也许可以使用“自动调整”（autofit）调整表格宽度，通常会得到完美的结果。

    自动调整到内容，然后窗口调整会得到最佳的列宽。

    表格 I - 因子载荷和共同度

    下图显示了一个 APA 格式的表格，其中结合了我们示例分析的因子载荷和共同度。

    如果你仔细观察 SPSS 输出，你会发现你_不能_简单地复制粘贴这些表格，并在 Excel 中将它们合并。这是因为因子载荷（模式矩阵 pattern matrix）表的变量顺序与共同度表不同。由于后者遵循你在语法中指定的变量顺序，因此最简单的解决方法是：

    • 确保输出中仅显示变量名称（而非标签）；
    • 将正确排序的模式矩阵复制粘贴到 Excel 中；
    • 将变量名称复制粘贴到 FACTOR 语法中并重新运行。

    提示：尝试将变量名称之间的换行符替换为空格，如下所示。

    在 SPSS 语法窗口中将换行符替换为空格

    此表格适用于探索性分析，但不适用于报告。

    表格 II - 总方差解释

    下面的屏幕截图显示了如何以 APA 格式报告特征值（Eigenvalues）表。

    相应的 SPSS 输出表非常接近此格式。但是，一个令人烦恼的问题是缺少百分号。

    如果我们复制粘贴到 Excel 并设置百分比格式，则 34.57 将转换为 3,457%。这是因为 Excel 将这些数字解释为比例，而不是 SPSS 所做的百分比点。最简单的解决方法是在复制粘贴到 Excel 之前，在 SPSS 中为这些列设置百分比格式。

    下面的 OUTPUT MODIFY 示例为输出窗口中的所有特征值表执行此操作。

    ***APPLY PERCENTAGE FORMAT TO PERCENT COLUMNS IN ALL TOTAL VARIANCE EXPLAINED TABLES.
**
output modify
/select tables
/tablecells select = ["% of Variance"] format = 'pct6.2'
/tablecells select = ["Cumulative %"] format = 'pct6.2'.

    完成此小修复后，你可以将此表格从 SPSS 复制粘贴到 Excel。我们现在可以轻松地进行一些最终调整（包括删除某些行和列），并将此表格复制粘贴到 WORD 中。
