理论模型

我们来看这个模型图，它描述了一个比较复杂的调节中介模型。 这里，X是自变量，Y是因变量，M是中介变量，W是调节变量。 箭头表示变量之间的影响关系。X会直接影响M和Y，M也会直接影响Y，W会直接影响M和Y。 此外，X和W的交互作用还会影响M和Y，M和W的交互作用也会影响Y。 这个模型主要研究的是：X对Y的影响如何通过M进行中介，以及W是如何调节X对M、M对Y以及X对Y的影响的。

数学模型

这张图展示了一个复杂的调节中介模型。模型包含四个主要变量：自变量X，中介变量M，调节变量W，以及因变量Y。

X会直接影响M (路径a1)，还会和W交互作用影响M (路径a3)。W也会直接影响M (路径a2)。

M会直接影响Y (路径b1)，并且M与W的交互作用也会影响Y (路径b2)。

X还会直接影响Y (路径c1’)，X与W的交互作用也会影响Y (路径c3’)。W也会直接影响Y (路径c2’)。

图中，这些带撇号的路径（c1’、c2’、c3’）表示X和W对Y的直接效应和交互效应。 每个路径上的字母代表了对应关系的强度，也就是路径系数。 XW和MW代表X和W的交互项。 总而言之，这个模型展示了X如何通过M影响Y，以及W是如何调节这个过程的。

数学公式1

Y = b0 + b1M + b2MW + c1'X + c2'W + c3'XW
M = a0 + a1X + a2W + a3XW

这是我们的初始模型方程。Y是因变量，X是自变量，M是中介变量，W是调节变量。第一个方程表示Y与M,W,X以及它们之间的交互项的关系；第二个方程表示M与X,W以及它们的交互项的关系。

数学公式2

Y = b0 + b1M + b2MW + c1'X + c2'W + c3'XW
M = a0 + a1X + a2W + a3XW
Y = b0 + b1(a0 + a1X + a2W + a3XW) + b2(a0 + a1X + a2W + a3XW)W + c1'X + c2'W + c3'XW

将第二个方程代入第一个方程，这一步是计算间接效应的关键。

数学公式3

Y = b0 + b1M + b2MW + c1'X + c2'W + c3'XW
M = a0 + a1X + a2W + a3XW
Y = b0 + b1(a0 + a1X + a2W + a3XW) + b2(a0 + a1X + a2W + a3XW)W + c1'X + c2'W + c3'XW
Y = b0 + a0b1 + a1b1X + a2b1W + a3b1XW + a0b2W + a1b2XW + a2b2W² + a3b2XW² + c1'X + c2'W + c3'XW

展开括号，得到一个更复杂的表达式。

数学公式4

Y = b0 + b1M + b2MW + c1'X + c2'W + c3'XW
M = a0 + a1X + a2W + a3XW
Y = b0 + b1(a0 + a1X + a2W + a3XW) + b2(a0 + a1X + a2W + a3XW)W + c1'X + c2'W + c3'XW
Y = b0 + a0b1 + a1b1X + a2b1W + a3b1XW + a0b2W + a1b2XW + a2b2W² + a3b2XW² + c1'X + c2'W + c3'XW
Y = (b0 + a0b1 + a2b1W + a0b2W + a2b2W² + c2'W) + (a1b1 + a3b1W + a1b2W + a3b2XW + c1' + c3'W)X

将公式整理成Y = a + bX的形式，以便分离出X的直接效应和间接效应。

数学公式5

Y = b0 + b1M + b2MW + c1'X + c2'W + c3'XW
M = a0 + a1X + a2W + a3XW
Y = b0 + b1(a0 + a1X + a2W + a3XW) + b2(a0 + a1X + a2W + a3XW)W + c1'X + c2'W + c3'XW
Y = b0 + a0b1 + a1b1X + a2b1W + a3b1XW + a0b2W + a1b2XW + a2b2W² + a3b2XW² + c1'X + c2'W + c3'XW
Y = (b0 + a0b1 + a2b1W + a0b2W + a2b2W² + c2'W) + (a1b1 + a3b1W + a1b2W + a3b2XW + c1' + c3'W)X
X 对 Y 的间接效应 (依赖于 W)：(a1 + a3W)(b1 + b2W)
X 对 Y 的直接效应 (依赖于 W)：c1' + c3'W

最终结果：X对Y的间接效应和直接效应的表达式。注意，两者都依赖于调节变量W。

代码解读1

USEVARIABLES = X1 X2 X3 X4 M1 M2 M3 M4 W1 W2 W3 W4 Y1 Y2 Y3 Y4;

首先，我们用USEVARIABLES语句指定分析中要用到的变量。X1到X4测量潜在自变量X，M1到M4测量潜在中介变量M，W1到W4测量潜在调节变量W，Y1到Y4测量潜在因变量Y。

代码解读2

USEVARIABLES = X1 X2 X3 X4 M1 M2 M3 M4 W1 W2 W3 W4 Y1 Y2 Y3 Y4;
ANALYSIS:
TYPE = GENERAL RANDOM;
ESTIMATOR = ML;
ALGORITHM = INTEGRATION;

接下来是ANALYSIS部分，我们指定模型类型为一般随机效应模型（TYPE = GENERAL RANDOM），使用最大似然估计法（ESTIMATOR = ML），并用积分算法进行估计（ALGORITHM = INTEGRATION）。这是处理潜在变量模型的标准做法。

代码解读3

USEVARIABLES = X1 X2 X3 X4 M1 M2 M3 M4 W1 W2 W3 W4 Y1 Y2 Y3 Y4;
ANALYSIS:
TYPE = GENERAL RANDOM;
ESTIMATOR = ML;
ALGORITHM = INTEGRATION;
MODEL:
X BY X1 X2 X3 X4;
M BY M1 M2 M3 M4;
W BY W1* W2 W3 W4;
Y BY Y1 Y2 Y3 Y4;

现在进入MODEL部分，我们定义测量模型。这四行代码分别定义了四个潜在变量X、M、W和Y的测量模型，X、M和Y都由四个指标变量来测量。注意，W BY W1* 表示W1的载荷固定为1，对W进行标准化。

代码解读4

USEVARIABLES = X1 X2 X3 X4 M1 M2 M3 M4 W1 W2 W3 W4 Y1 Y2 Y3 Y4;
ANALYSIS:
TYPE = GENERAL RANDOM;
ESTIMATOR = ML;
ALGORITHM = INTEGRATION;
MODEL:
X BY X1 X2 X3 X4;
M BY M1 M2 M3 M4;
W BY W1* W2 W3 W4;
Y BY Y1 Y2 Y3 Y4;
W@1;

这一行代码将调节变量W的方差固定为1，同样也是为了标准化。

代码解读5

USEVARIABLES = X1 X2 X3 X4 M1 M2 M3 M4 W1 W2 W3 W4 Y1 Y2 Y3 Y4;
ANALYSIS:
TYPE = GENERAL RANDOM;
ESTIMATOR = ML;
ALGORITHM = INTEGRATION;
MODEL:
X BY X1 X2 X3 X4;
M BY M1 M2 M3 M4;
W BY W1* W2 W3 W4;
Y BY Y1 Y2 Y3 Y4;
W@1;
MW | M XWITH W;
XW | X XWITH W;

这两行代码定义了潜在交互项。MW代表M和W的交互作用，XW代表X和W的交互作用。这部分定义了模型的调节效应。

代码解读6

USEVARIABLES = X1 X2 X3 X4 M1 M2 M3 M4 W1 W2 W3 W4 Y1 Y2 Y3 Y4;
ANALYSIS:
TYPE = GENERAL RANDOM;
ESTIMATOR = ML;
ALGORITHM = INTEGRATION;
MODEL:
X BY X1 X2 X3 X4;
M BY M1 M2 M3 M4;
W BY W1* W2 W3 W4;
Y BY Y1 Y2 Y3 Y4;
W@1;
MW | M XWITH W;
XW | X XWITH W;
Y ON M (b1);
Y ON MW (b2);
Y ON X (cdash1);
Y ON W (cdash2);
Y ON XW (cdash3);

这些语句定义了结构方程模型中Y的回归方程。b1, b2, cdash1, cdash2, cdash3 分别代表相应的回归系数。我们可以看到Y受M、MW、X、W、XW的影响。

代码解读7

USEVARIABLES = X1 X2 X3 X4 M1 M2 M3 M4 W1 W2 W3 W4 Y1 Y2 Y3 Y4;
ANALYSIS:

Y ON MW (b2);
Y ON X (cdash1);
Y ON W (cdash2);
Y ON XW (cdash3);

M ON W (a2);
M ON XW (a3);

接下来是M的回归方程，M受X, W, XW的影响，a1, a2, a3是对应的回归系数。

NEW(LOW_W MED_W HIGH_W ...);

MODEL CONSTRAINT部分开始，我们定义新的参数来计算在不同调节变量水平下的间接效应、直接效应和总效应。这里定义了W的三个水平：-1标准差，均值和+1标准差。

这里是一系列约束方程，用于计算不同W水平下，X对Y的间接效应（IND_LOWW等）、调节间接效应(IMM_LOW等)、直接效应(DIR_LOWW等)以及总效应(TOT_LOWW等)。这些公式根据调节中介效应的模型推导而来。

USEVARIABLES = X1 X2 X3 X4 M1 M2 M3 M4 W1 W2 W3 W4 Y1 Y2 Y3 Y4;

TYPE = GENERAL RANDOM;
ESTIMATOR = ML;
ALGORITHM = INTEGRATION;
