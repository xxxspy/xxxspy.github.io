理论模型

我们来看这个模型图，它展示了一个包含中介和调节效应的复杂模型。X是自变量，Y是因变量，M是中介变量，W和Z是调节变量。 箭头表示变量间的影响方向。X不仅直接影响Y，还通过M间接影响Y。 W和Z分别调节X对M的影响，而W还调节M对Y的影响。简单来说，就是X影响M，M影响Y，X直接影响Y，W和Z改变了X影响M的强度，W还改变了M影响Y的强度。 Z只影响中介变量M，不影响其他变量。 接下来，我们将用数学模型和代码来更精确地描述这个模型。

数学模型

这张图展示了我们模型的数学结构。 可以看到，我们考虑了一个自变量X，一个因变量Y，以及一个中介变量M。 X会通过不同的路径影响M，而M又会影响Y。 更重要的是，我们还引入了两个调节变量W和Z，以及它们的交互项XW和XZ。这些调节变量会影响X到M，以及M到Y的路径。此外，我们还考虑了M和W的交互项MW对Y的影响。 通过这些路径，我们可以分析X对Y的直接效应和间接效应，以及W和Z的调节作用。 这是一个比较复杂的模型，但它能更全面地解释变量之间的关系。

数学公式1

Y = b0 + b1M + b2W + b3MW + c'X

这是我们的模型方程，其中Y是因变量，X是自变量，M是中介变量，W是调节变量，c’代表X对Y的直接效应。

数学公式2

Y = b0 + b1M + b2W + b3MW + c'X
M = a0 + a1X + a2W + a3Z + a4XW + a5XZ

这是中介变量M的方程，它受到自变量X、调节变量W和Z以及它们与X的交互项的影响。

数学公式3

Y = b0 + b1M + b2W + b3MW + c'X
M = a0 + a1X + a2W + a3Z + a4XW + a5XZ
Y = b0 + b1(a0 + a1X + a2W + a3Z + a4XW + a5XZ) + b2W + b3(a0 + a1X + a2W + a3Z + a4XW + a5XZ)W + c'X

将第二个方程代入第一个方程，得到一个包含X, W, Z的联合方程。

数学公式4

Y = b0 + b1M + b2W + b3MW + c'X
M = a0 + a1X + a2W + a3Z + a4XW + a5XZ
Y = b0 + b1(a0 + a1X + a2W + a3Z + a4XW + a5XZ) + b2W + b3(a0 + a1X + a2W + a3Z + a4XW + a5XZ)W + c'X
Y = b0 + a0b1 + a1b1X + a2b1W + a3b1Z + a4b1XW + a5b1XZ + b2W + a0b3W + a1b3XW + a2b3WW + a3b3ZW + a4b3XWW + a5b3XZW + c'X

展开括号，得到一个更复杂的方程。

数学公式5

Y = b0 + b1M + b2W + b3MW + c'X
M = a0 + a1X + a2W + a3Z + a4XW + a5XZ
Y = b0 + b1(a0 + a1X + a2W + a3Z + a4XW + a5XZ) + b2W + b3(a0 + a1X + a2W + a3Z + a4XW + a5XZ)W + c'X
Y = b0 + a0b1 + a1b1X + a2b1W + a3b1Z + a4b1XW + a5b1XZ + b2W + a0b3W + a1b3XW + a2b3WW + a3b3ZW + a4b3XWW + a5b3XZW + c'X
Y = (b0 + a0b1 + a2b1W + a3b1Z + b2W + a0b3W + a2b3WW + a3b3ZW) + (a1b1 + a4b1W + a5b1Z + a1b3W + a4b3WW + a5b3ZW + c')X

整理成Y = a + bX的形式，方便我们提取X对Y的间接效应和直接效应。

数学公式6

Y = b0 + b1M + b2W + b3MW + c'X
M = a0 + a1X + a2W + a3Z + a4XW + a5XZ
Y = b0 + b1(a0 + a1X + a2W + a3Z + a4XW + a5XZ) + b2W + b3(a0 + a1X + a2W + a3Z + a4XW + a5XZ)W + c'X
Y = b0 + a0b1 + a1b1X + a2b1W + a3b1Z + a4b1XW + a5b1XZ + b2W + a0b3W + a1b3XW + a2b3WW + a3b3ZW + a4b3XWW + a5b3XZW + c'X
Y = (b0 + a0b1 + a2b1W + a3b1Z + b2W + a0b3W + a2b3WW + a3b3ZW) + (a1b1 + a4b1W + a5b1Z + a1b3W + a4b3WW + a5b3ZW + c')X
条件于 W 和 Z 的 X 对 Y 的间接效应为：(a1 + a4W + a5Z)(b1 + b3W)

这是条件间接效应的公式，它显示了在不同W和Z水平下，X通过M对Y的影响。注意，这是包含两个调节变量的间接效应。

数学公式7

Y = b0 + b1M + b2W + b3MW + c'X
M = a0 + a1X + a2W + a3Z + a4XW + a5XZ
Y = b0 + b1(a0 + a1X + a2W + a3Z + a4XW + a5XZ) + b2W + b3(a0 + a1X + a2W + a3Z + a4XW + a5XZ)W + c'X
Y = b0 + a0b1 + a1b1X + a2b1W + a3b1Z + a4b1XW + a5b1XZ + b2W + a0b3W + a1b3XW + a2b3WW + a3b3ZW + a4b3XWW + a5b3XZW + c'X
Y = (b0 + a0b1 + a2b1W + a3b1Z + b2W + a0b3W + a2b3WW + a3b3ZW) + (a1b1 + a4b1W + a5b1Z + a1b3W + a4b3WW + a5b3ZW + c')X
条件于 W 和 Z 的 X 对 Y 的间接效应为：(a1 + a4W + a5Z)(b1 + b3W)
X 对 Y 的直接效应为：c'

这是X对Y的直接效应，它不受中介变量M的影响。

代码解读1

USEVARIABLES = X1 X2 X3 X4 M1 M2 M3 M4 W1 W2 W3 W4 Z1 Z2 Z3 Z4 Y1 Y2 Y3 Y4;

首先，我们定义了模型中用到的所有变量。X1到X4代表自变量X的四个指标，M1到M4是中介变量M的指标，以此类推，W和Z是两个调节变量，Y是因变量。

代码解读2

USEVARIABLES = X1 X2 X3 X4 M1 M2 M3 M4 W1 W2 W3 W4 Z1 Z2 Z3 Z4 Y1 Y2 Y3 Y4;
ANALYSIS:
TYPE = GENERAL RANDOM;
ESTIMATOR = ML;
ALGORITHM = INTEGRATION;

接下来，我们设置分析方法。我们采用广义随机效应模型，使用最大似然估计法，并用数值积分法处理潜在变量，保证模型的准确性。

代码解读3

USEVARIABLES = X1 X2 X3 X4 M1 M2 M3 M4 W1 W2 W3 W4 Z1 Z2 Z3 Z4 Y1 Y2 Y3 Y4;
ANALYSIS:
TYPE = GENERAL RANDOM;
ESTIMATOR = ML;
ALGORITHM = INTEGRATION;
MODEL:
X BY X1 X2 X3 X4;
M BY M1 M2 M3 M4;
W BY W1* W2 W3 W4;
Z BY Z1* Z2 Z3 Z4;
Y BY Y1 Y2 Y3 Y4;

这部分定义了测量模型，将潜变量X、M、W、Z、Y与它们对应的观测变量联系起来。注意，W1和Z1后加了星号，表示它们的方差被固定为1，进行标准化。

代码解读4

ANALYSIS:

Z BY Z1* Z2 Z3 Z4;
Y BY Y1 Y2 Y3 Y4;
W@1;  Z@1;

这里再次强调，我们将W和Z的方差固定为1，确保它们被标准化。

代码解读5

MW | M XWITH W;
XW | X XWITH W;
XZ | X XWITH Z;

这一部分定义了潜变量之间的交互效应。MW表示M和W的交互作用，XW表示X和W的交互作用，XZ表示X和Z的交互作用，这些交互项体现了调节效应。

这是结构模型的一部分，描述了Y与M、W、MW和X之间的关系，b1、b2、b3和cdash代表相应的路径系数。
