各位好，今天我将向大家介绍一个关于复杂中介调节模型的Mplus实现教程。我们将从理论基础出发，逐步深入到数学模型、数学推导以及最终的代码实现。首先，我们会详细讲解这个模型的理论框架，明确各个变量之间的关系，并解释它们的作用机制。接着，我们会将这个理论模型用具体的数学方程来表达，定义各路径之间的关系。然后，我们将会展示如何从数学模型推导出我们需要检验的直接效应、间接效应和调节效应的公式。最后，我们会一步步解释Mplus代码的各个部分，包括如何定义变量、设定模型、进行参数估计以及如何解读结果。通过本教程，您将掌握这个复杂模型的Mplus实现方法，并能够将其应用于您自己的研究中。

理论模型

我们来看，这个模型展示了几个变量之间的关系。W和X都影响着M1、M2和Y。M1影响着M2和Y，而M2又影响着Y。箭头表示变量之间可能存在的影响关系。

数学模型

这张图展示了一个复杂的统计模型，它揭示了X、W和XW如何通过多条路径影响Y。XW通过a4这条路影响M1，同时通过a6影响M2，它还通过c3’这条路直接影响Y。W通过a3影响M1，通过a5影响M2，并且通过c2’直接影响Y。X通过a1影响M2，通过a2影响M2，并且通过c1’直接影响Y。M1通过d1影响M2，再通过b1影响Y，而M2通过b2影响Y。这个图详细展示了各个变量之间错综复杂的关系。

数学公式1

模型方程:

Y = b0 + b1M1 + b2M2 + c1'X + c2'W + c3'XW
M1 = a01 + a1X + a3W + a4XW
M2 = a02 + a2X + a5W + a6XW + d1M1

首先，我们定义了三个模型方程。第一个方程，Y，是结果变量，它受到两个中介变量M1和M2，以及自变量X和调节变量W的影响。M1受到X和W的直接影响，以及它们交互项的影响。M2不仅受到X和W的影响，还受到M1的影响。

数学公式2

模型方程:

Y = b0 + b1M1 + b2M2 + c1'X + c2'W + c3'XW
M1 = a01 + a1X + a3W + a4XW
M2 = a02 + a2X + a5W + a6XW + d1M1
代入和展开:

Y = b0 + b1(a01 + a1X + a3W + a4XW) + b2(a02 + a2X + a5W + a6XW + d1(a01 + a1X + a3W + a4XW)) + c1'X + c2'W + c3'XW

Y = b0 + a01b1 + a1b1X + a3b1W + a4b1XW + a02b2 + a2b2X + a5b2W + a6b2XW + a01d1b2 + a1d1b2X + a3d1b2W + a4d1b2XW + c1'X + c2'W + c3'XW

接下来，我们将M1和M2的表达式代入到Y的方程中，然后展开所有括号。这个步骤的目的是为了将X对Y的整体影响分解出来，以便后续分析。

数学公式3

模型方程:

Y = b0 + b1M1 + b2M2 + c1'X + c2'W + c3'XW
M1 = a01 + a1X + a3W + a4XW
M2 = a02 + a2X + a5W + a6XW + d1M1
代入和展开:

Y = b0 + b1(a01 + a1X + a3W + a4XW) + b2(a02 + a2X + a5W + a6XW + d1(a01 + a1X + a3W + a4XW)) + c1'X + c2'W + c3'XW

Y = b0 + a01b1 + a1b1X + a3b1W + a4b1XW + a02b2 + a2b2X + a5b2W + a6b2XW + a01d1b2 + a1d1b2X + a3d1b2W + a4d1b2XW + c1'X + c2'W + c3'XW
合并同类项:

Y = (b0 + a01b1 + a3b1W + a02b2 + a5b2W + a01d1b2 + a3d1b2W + c2'W) + (a1b1 + a4b1W + a2b2 + a6b2W + a1d1b2 + a4d1b2W + c1' + c3'W)X

现在，我们将方程中的各项重新排列，把所有不含X的项合并为一个常数项，所有含X的项合并为X的系数。这样，我们得到了一个清晰的Y关于X的线性方程形式，方便提取效应系数。

数学公式4

模型方程:

Y = b0 + b1M1 + b2M2 + c1'X + c2'W + c3'XW
M1 = a01 + a1X + a3W + a4XW
M2 = a02 + a2X + a5W + a6XW + d1M1
代入和展开:

Y = b0 + b1(a01 + a1X + a3W + a4XW) + b2(a02 + a2X + a5W + a6XW + d1(a01 + a1X + a3W + a4XW)) + c1'X + c2'W + c3'XW

Y = b0 + a01b1 + a1b1X + a3b1W + a4b1XW + a02b2 + a2b2X + a5b2W + a6b2XW + a01d1b2 + a1d1b2X + a3d1b2W + a4d1b2XW + c1'X + c2'W + c3'XW
合并同类项:

Y = (b0 + a01b1 + a3b1W + a02b2 + a5b2W + a01d1b2 + a3d1b2W + c2'W) + (a1b1 + a4b1W + a2b2 + a6b2W + a1d1b2 + a4d1b2W + c1' + c3'W)X
结果:

X对Y的三个有条件间接效应 (Conditional indirect effects):

通过 M1： (a1 + a4W)b1
通过 M2： (a2 + a6W)b2
通过 M1 和 M2： (a1 + a4W)d1b2

X对Y的一个有条件直接效应 (Conditional direct effect)：

c1' + c3'W

最后，我们从合并同类项后的方程中提取出关键的效应。X对Y的影响分为直接效应和间接效应。间接效应又可以分解为通过M1，通过M2，以及通过M1和M2的效应。这些效应都是有条件的，它们的大小会受到调节变量W的影响。直接效应也是一个有条件的影响，也会随着W变化。

代码解读1

USEVARIABLES = X M1 M2 W Y XW;
! Create interaction term

第一步，我们使用USEVARIABLES命令来声明分析中要用到的变量。包括预测变量X，两个中介变量M1和M2，调节变量W，结果变量Y，以及X和W的交互项XW。感叹号开头的是注释，这里提示我们接下来要创建交互项。

代码解读2

USEVARIABLES = X M1 M2 W Y XW;
! Create interaction term
DEFINE:

 XW = X*W;

第二步，在DEFINE模块中，我们使用X乘以W，创建了一个新的变量XW，它代表预测变量X和调节变量W的交互作用。

代码解读3

USEVARIABLES = X M1 M2 W Y XW;
! Create interaction term
DEFINE:

 XW = X*W;
ANALYSIS:

 TYPE = GENERAL;
 ESTIMATOR = ML;
 BOOTSTRAP = 10000;

第三步，在ANALYSIS模块，我们指定分析的类型和参数。TYPE设为GENERAL，表示使用一般模型；ESTIMATOR设为ML，表示使用最大似然估计方法；BOOTSTRAP设为10000，表示使用10000次自助抽样来计算参数的标准误差和置信区间。

代码解读4

! Create interaction term
DEFINE:

ANALYSIS:

 TYPE = GENERAL;
 ESTIMATOR = ML;

MODEL:

 Y ON M2 (b2);

 M1 ON X (a1);
 M1 ON W (a3);
 M1 ON XW (a4);
 M2 ON X (a2);
 M2 ON W (a5);
 M2 ON XW (a6);
 M2 ON M1 (d1);

第四步，MODEL模块定义了模型中的路径关系。ON表示因果关系。例如，Y ON M1 (b1)表示M1对Y的回归路径，回归系数标记为b1。这里定义了所有变量间的回归关系，包括直接效应、中介效应以及交互效应。
