大家好，欢迎来到本教程。 这份PPT介绍了如何使用Mplus构建一个复杂的调节中介模型。 这个模型包含一个或多个中介变量，以及四个调节变量，并且包含了丰富的交互作用项。

首先，我们会讲解模型的理论基础，也就是这个模型想要回答什么问题，以及各个变量之间的关系是如何被假设的。 这部分内容会帮助大家理解模型背后的逻辑。

接下来，我们会用数学公式来表达这个复杂的调节中介模型。 这部分会比较抽象，但会为我们后面Mplus代码的编写打下基础。

然后，我们会逐步推导这个数学模型中各个参数的含义以及它们之间的关系。 这部分重点在于解释如何计算条件间接效应和直接效应。

最后，我们会详细解读Mplus的代码，解释每一行代码的含义以及如何运行和解读结果。 我们会重点关注如何设置模型、提取参数以及进行相关的统计检验。 通过本教程，希望大家能够快速掌握如何使用Mplus构建和分析这种复杂的调节中介模型。 如有任何疑问，欢迎随时提出。

理论模型

我们来看这张图，它展示了一个比较复杂的模型。 这是一个调节中介模型，包含一个自变量X，一个中介变量M，一个因变量Y，以及四个调节变量W、Z、V、Q。 图中的箭头表示变量之间的关系，比如X影响M，M影响Y，X也直接影响Y。 更重要的是，W和Z调节着X对M和Y的影响，V和Q则调节着M对Y的影响，并且V和Q本身也直接影响Y。 接下来我们会用数学模型来更精确地描述这些关系，并用Mplus来分析。

数学模型

这张图展示了一个复杂的调节中介模型。我们来看一下各个变量：X是自变量，Y是因变量，M是中介变量，W、Z、V、Q是调节变量。 X通过不同的路径影响M和Y，M也影响Y。 W和Z调节X对M和Y的影响，V和Q调节M对Y的影响。图中还包含了多个交互项，例如XW、XZ、WZ、XWZ等等，它们代表了变量之间的交互作用。这个模型很复杂，因为它考虑了多个变量以及它们之间多种复杂的交互关系，用于研究这些变量如何共同影响最终结果Y。

数学公式1

Y = b0 + b1M + b2V + b3Q + b4MV + b5MQ + c1'X + c2'W + c3'Z + c4'XW + c5'XZ + c6'WZ + c7'XWZ

这是我们的初始模型方程，其中Y是因变量，M, V, Q, X, W, Z是自变量。

数学公式2

Y = b0 + b1M + b2V + b3Q + b4MV + b5MQ + c1'X + c2'W + c3'Z + c4'XW + c5'XZ + c6'WZ + c7'XWZ
M = a0 + a1X + a2W + a3Z + a4XW + a5XZ + a6WZ + a7XWZ

这是第二个方程，描述了自变量M与其他自变量之间的关系。

数学公式3

Y = b0 + b1M + b2V + b3Q + b4MV + b5MQ + c1'X + c2'W + c3'Z + c4'XW + c5'XZ + c6'WZ + c7'XWZ
M = a0 + a1X + a2W + a3Z + a4XW + a5XZ + a6WZ + a7XWZ
Y = b0 + b1(a0 + a1X + a2W + a3Z + a4XW + a5XZ + a6WZ + a7XWZ) + b2V + b3Q + b4(a0 + a1X + a2W + a3Z + a4XW + a5XZ + a6WZ + a7XWZ)V + b5(a0 + a1X + a2W + a3Z + a4XW + a5XZ + a6WZ + a7XWZ)Q + c1'X + c2'W + c3'Z + c4'XW + c5'XZ + c6'WZ + c7'XWZ

将第二个方程代入第一个方程，得到一个包含X, W, Z, V, Q的复杂表达式。

Y = b0 + b1M + b2V + b3Q + b4MV + b5MQ + c1'X + c2'W + c3'Z + c4'XW + c5'XZ + c6'WZ + c7'XWZ
M = a0 + a1X + a2W + a3Z + a4XW + a5XZ + a6WZ + a7XWZ
Y = b0 + b1(a0 + a1X + a2W + a3Z + a4XW + a5XZ + a6WZ + a7XWZ) + b2V + b3Q + b4(a0 + a1X + a2W + a3Z + a4XW + a5XZ + a6WZ + a7XWZ)V + b5(a0 + a1X + a2W + a3Z + a4XW + a5XZ + a6WZ + a7XWZ)Q + c1'X + c2'W + c3'Z + c4'XW + c5'XZ + c6'WZ + c7'XWZ
Y = b0 + a0b1 + a1b1X + a2b1W + a3b1Z + a4b1XW + a5b1XZ + a6b1WZ + a7b1XWZ + b2V + b3Q + a0b4V + a1b4XV + a2b4WV + a3b4ZV + a4b4XWV + a5b4XZV + a6b4WZV + a7b4XWZV + a0b5Q + a1b5XQ + a2b5WQ + a3b5ZQ + a4b5XWQ + a5b5XZQ + a6b5WZQ + a7b5XWZQ + c1'X + c2'W + c3'Z + c4'XW + c5'XZ + c6'WZ + c7'XWZ

展开括号，得到一个更复杂的表达式。

数学公式5

Y = b0 + b1M + b2V + b3Q + b4MV + b5MQ + c1'X + c2'W + c3'Z + c4'XW + c5'XZ + c6'WZ + c7'XWZ
M = a0 + a1X + a2W + a3Z + a4XW + a5XZ + a6WZ + a7XWZ
Y = b0 + b1(a0 + a1X + a2W + a3Z + a4XW + a5XZ + a6WZ + a7XWZ) + b2V + b3Q + b4(a0 + a1X + a2W + a3Z + a4XW + a5XZ + a6WZ + a7XWZ)V + b5(a0 + a1X + a2W + a3Z + a4XW + a5XZ + a6WZ + a7XWZ)Q + c1'X + c2'W + c3'Z + c4'XW + c5'XZ + c6'WZ + c7'XWZ
Y = b0 + a0b1 + a1b1X + a2b1W + a3b1Z + a4b1XW + a5b1XZ + a6b1WZ + a7b1XWZ + b2V + b3Q + a0b4V + a1b4XV + a2b4WV + a3b4ZV + a4b4XWV + a5b4XZV + a6b4WZV + a7b4XWZV + a0b5Q + a1b5XQ + a2b5WQ + a3b5ZQ + a4b5XWQ + a5b5XZQ + a6b5WZQ + a7b5XWZQ + c1'X + c2'W + c3'Z + c4'XW + c5'XZ + c6'WZ + c7'XWZ
Y = (b0 + a0b1 + a2b1W + a3b1Z + a6b1WZ + b2V + b3Q + a0b4V + a2b4WV + a3b4ZV + a6b4WZV + a0b5Q + a2b5WQ + a3b5ZQ + a6b5WZQ + c2'W + c3'Z + c6'WZ) + (a1b1 + a4b1W + a5b1Z + a7b1WZ + a1b4V + a4b4WV + a5b4ZV + a7b4WZV + a1b5Q + a4b5WQ + a5b5ZQ + a7b5WZQ + c1' + c4'W + c5'Z + c7'WZ)X

将含有X的项和不含X的项分别归类，整理成Y = a + bX的形式。

数学公式6

Y = b0 + b1M + b2V + b3Q + b4MV + b5MQ + c1'X + c2'W + c3'Z + c4'XW + c5'XZ + c6'WZ + c7'XWZ
M = a0 + a1X + a2W + a3Z + a4XW + a5XZ + a6WZ + a7XWZ
Y = b0 + b1(a0 + a1X + a2W + a3Z + a4XW + a5XZ + a6WZ + a7XWZ) + b2V + b3Q + b4(a0 + a1X + a2W + a3Z + a4XW + a5XZ + a6WZ + a7XWZ)V + b5(a0 + a1X + a2W + a3Z + a4XW + a5XZ + a6WZ + a7XWZ)Q + c1'X + c2'W + c3'Z + c4'XW + c5'XZ + c6'WZ + c7'XWZ
Y = b0 + a0b1 + a1b1X + a2b1W + a3b1Z + a4b1XW + a5b1XZ + a6b1WZ + a7b1XWZ + b2V + b3Q + a0b4V + a1b4XV + a2b4WV + a3b4ZV + a4b4XWV + a5b4XZV + a6b4WZV + a7b4XWZV + a0b5Q + a1b5XQ + a2b5WQ + a3b5ZQ + a4b5XWQ + a5b5XZQ + a6b5WZQ + a7b5XWZQ + c1'X + c2'W + c3'Z + c4'XW + c5'XZ + c6'WZ + c7'XWZ
Y = (b0 + a0b1 + a2b1W + a3b1Z + a6b1WZ + b2V + b3Q + a0b4V + a2b4WV + a3b4ZV + a6b4WZV + a0b5Q + a2b5WQ + a3b5ZQ + a6b5WZQ + c2'W + c3'Z + c6'WZ) + (a1b1 + a4b1W + a5b1Z + a7b1WZ + a1b4V + a4b4WV + a5b4ZV + a7b4WZV + a1b5Q + a4b5WQ + a5b5ZQ + a7b5WZQ + c1' + c4'W + c5'Z + c7'WZ)X
X对Y的间接效应（在W, Z, V, Q条件下）：(a1 + a4W + a5Z + a7WZ)(b1 + b4V + b5Q)

这是X对Y的间接效应，通过M进行传递。

Y = b0 + a0b1 + a1b1X + a2b1W + a3b1Z + a4b1XW + a5b1XZ + a6b1WZ + a7b1XWZ + b2V + b3Q + a0b4V + a1b4XV + a2b4WV + a3b4ZV + a4b4XWV + a5b4XZV + a6b4WZV + a7b4XWZV + a0b5Q + a1b5XQ + a2b5WQ + a3b5ZQ + a4b5XWQ + a5b5XZQ + a6b5WZQ + a7b5XWZQ + c1'X + c2'W + c3'Z + c4'XW + c5'XZ + c6'WZ + c7'XWZ
Y = (b0 + a0b1 + a2b1W + a3b1Z + a6b1WZ + b2V + b3Q + a0b4V + a2b4WV + a3b4ZV + a6b4WZV + a0b5Q + a2b5WQ + a3b5ZQ + a6b5WZQ + c2'W + c3'Z + c6'WZ) + (a1b1 + a4b1W + a5b1Z + a7b1WZ + a1b4V + a4b4WV + a5b4ZV + a7b4WZV + a1b5Q + a4b5WQ + a5b5ZQ + a7b5WZQ + c1' + c4'W + c5'Z + c7'WZ)X
X对Y的间接效应（在W, Z, V, Q条件下）：(a1 + a4W + a5Z + a7WZ)(b1 + b4V + b5Q)
X对Y的直接效应（在W, Z条件下）：c1' + c4'W + c5'Z + c7'WZ

这是X对Y的直接效应，不通过M进行传递。

代码解读1

USEVARIABLES = X M W Z V Q Y XW XZ WZ MV MQ XWZ;

首先，我们定义了分析中用到的所有变量。X是自变量，Y是因变量，M是中介变量，W、Z、V、Q是调节变量。XW、XZ等等，这些都是我们预先计算好的交互项，一会儿我们会看到它们是怎么定义的。

代码解读2

USEVARIABLES = X M W Z V Q Y XW XZ WZ MV MQ XWZ;
DEFINE:
 MQ = M*Q;
 MV = M*V;
 XW = X*W;
 XZ = X*Z;
 WZ = W*Z;
 XWZ = X*W*Z;

这里我们用DEFINE语句定义了前面提到的交互项。例如，MQ就是M和Q的乘积，它代表了M和Q的交互作用。其他的交互项也是类似的计算方式。

代码解读3

USEVARIABLES = X M W Z V Q Y XW XZ WZ MV MQ XWZ;
DEFINE:
 MQ = M*Q;
 MV = M*V;
 XW = X*W;
 XZ = X*Z;
 WZ = W*Z;
 XWZ = X*W*Z;
ANALYSIS:
 TYPE = GENERAL;
 ESTIMATOR = ML;
 BOOTSTRAP = 10000;

接下来，我们设定分析的类型、估计方法和bootstrap抽样次数。这里我们选择了一般性模型，使用最大似然估计法，并进行了10000次bootstrap抽样，以获得更精确的置信区间。

这是模型的结构定义。我们定义了Y与各个变量之间的关系，以及M与各个变量的关系，包括直接效应和交互效应。例如，‘Y ON M (b1)’ 表示Y受M的影响，b1是路径系数。模型包含了大量的交互项，使得模型能够考察复杂的调节中介效应。
