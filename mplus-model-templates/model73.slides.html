来自图书《MPlus中介调节模型》

调节中介模型分析教程结构

  • 理论模型：详细阐述模型中变量间的概念关系，以及中介、调节效应的理论基础。
  • 数学模型：将理论模型转化为具体的数学方程式，明确各变量之间的数学关系，包括主效应、交互效应。
  • 数学推导：推导计算间接效应和条件效应的代数公式，为后续模型检验提供理论依据。
  • 代码解读：详细解读 Mplus 代码，包括如何构建交互项、使用约束检验条件间接效应，以及绘制条件间接效应图。

大家好，今天我们来介绍一个关于调节中介模型分析的教程。这个教程结构严谨，旨在帮助大家深入理解并应用复杂的调节中介模型。首先，我们会从理论层面出发，详细讲解模型中变量之间的概念关系，以及中介效应和调节效应的理论基础。接着，我们将把理论模型转化为具体的数学方程式，明确各变量之间主效应和交互效应的数学关系。然后，我们会进行数学推导，推导出计算间接效应和条件效应的代数公式，为后续的模型检验提供理论依据。最后，我们会详细解读Mplus代码，包括如何构建交互项，如何使用约束检验条件间接效应，以及如何绘制条件间接效应图。通过这个教程，大家可以全面掌握调节中介模型的分析方法，并将其应用于实际研究中。

理论模型

我们来看，这个模型中有五个变量，分别是X、M、W、Z和Y。箭头表示变量之间的影响关系。例如，X直接影响M和Y，W也直接影响M和Y，Z也直接影响M和Y，M又会影响Y。此外，W和Z还会互相作用影响M和Y。X，W和Z也会相互影响M和Y。

数学模型

这张图展示了一个复杂的统计模型。其中，X、W和Z这三个变量会影响中间变量M。X、W、Z，以及它们的组合XW、XZ和WZ，还有更复杂的组合XWZ，都指向M。M又会影响结果变量Y。此外，X、W、Z、XW、XZ、WZ、XWZ，还有M与W的组合MW、M与Z的组合MZ，以及M、W、Z三者的组合MWZ，也都直接指向Y。图中箭头上的a1到a7，表示X、W、Z和它们的组合对M的影响大小。b1到b4，表示M和它的组合对Y的影响大小。c1’到c7’，表示X、W、Z和它们的组合对Y的直接影响。

数学公式1

Y = b0 + b1M + b2MW + b3MZ + b4MWZ + c1'X + c2W + c3'Z + c4'XW + c5'XZ + c6'WZ + c7'XWZ
M = a0 + a1X + a2W + a3Z + a4XW + a5XZ + a6WZ + a7XWZ

首先，我们有两个方程。第一个方程描述了因变量 Y 与中介变量 M 以及调节变量 W 和 Z 的关系。第二个方程描述了中介变量 M 与自变量 X 以及调节变量 W 和 Z 的关系。我们的目标是将 M 的表达式代入 Y 的表达式，以便将 Y 表示为 X 的函数。

Y = b0 + b1(a0 + a1X + a2W + a3Z + a4XW + a5XZ + a6WZ + a7XWZ) + b2(a0 + a1X + a2W + a3Z + a4XW + a5XZ + a6WZ + a7XWZ)W + b3(a0 + a1X + a2W + a3Z + a4XW + a5XZ + a6WZ + a7XWZ)Z + b4(a0 + a1X + a2W + a3Z + a4XW + a5XZ + a6WZ + a7XWZ)WZ + c1'X + c2W + c3'Z + c4'XW + c5'XZ + c6'WZ + c7'XWZ

接下来，我们将 M 的表达式代入 Y 的表达式中。这里可以看到，我们将M替换为了包含X,W,Z的复杂表达式。

最后，我们提取出 X 对 Y 的间接效应和直接效应。间接效应是 X 通过中介变量 M 对 Y 的影响，它取决于调节变量 W 和 Z 的条件。直接效应是 X 对 Y 的直接影响，同样取决于调节变量 W 和 Z 的条件。间接效应可以进一步简化为两个多项式相乘的形式。

代码解读1

USEVARIABLES = X M W Z Y XW XZ WZ MW MZ XWZ MWZ;
! Create interaction terms

! Note that they have to be placed at end of USEVARIABLES subcommand above
DEFINE:

 MW = M*W;
 MZ = M*Z;
 XW = X*W;
 XZ = X*Z;
 WZ = W*Z;
 MWZ = M*W*Z;
 XWZ = X*W*Z;

第一步，我们定义模型中使用的变量。USEVARIABLES 声明了所有变量，包括自变量X，中介变量M，调节变量W和Z，以及因变量Y。同时还包括了变量之间的交互项，比如XW，XZ等等。接下来，在DEFINE部分，我们创建这些交互项，通过将相应的变量相乘得到。注意，这些交互项必须在USEVARIABLES命令之后定义。

第二步，我们设置分析参数。ANALYSIS部分定义了分析类型为一般模型，使用最大似然估计方法，并设置10000次自助抽样。在MODEL部分，我们定义模型的具体路径。例如，[Y] (b0) 定义了Y的截距为b0，Y ON M (b1)定义了M对Y的回归系数为b1，以此类推，我们为所有路径和截距都指定了名称。
