1.1 引言

欢迎来到 Mplus 中介、调节和调节中介模型教程！本教程旨在为您提供使用 Mplus 软件进行复杂统计模型分析的实用指导。在这里，您将找到各种模型配置的 Mplus 代码示例，包括中介效应、调节效应和调节中介效应的组合。

本教程的核心内容在于展示 Mplus 代码，以便您能够测试不同的模型结构。每个模型都配有概念和统计模型图、模型方程以及最重要的 Mplus 代码，包括 DEFINE:，ANALYSIS:，MODEL: 和 OUTPUT: 等关键命令，以及列出变量的 USEVARIABLES: 子命令。您可以通过调整 DATA: 和 VARIABLE: 命令，并将示例中的变量名称替换为您自己的数据中的变量名，从而将这些示例代码应用于您的研究。

本教程中的大多数模型类型（以及编号系统）与 Andrew Hayes 在其 PROCESS 宏（用于 SPSS 和 SAS）的 v2 和 v3 版本文档以及其开创性著作《中介、调节和条件过程分析导论》（Hayes，2013，2017）中列出和索引的配置相对应。我强烈建议您在进行此类分析之前阅读此书，作为入门指南。对于 PROCESS v3（及其书籍的第二版），Andrew 删除了一些具有 3 个或以上调节变量的模型，但我在本教程中保留了这些模型的代码。

重要提示： Mplus 提供了完全灵活的建模环境，允许您测试无限的调节和中介配置，而不仅仅是 PROCESS 模型类型。 具体来说，与 PROCESS 宏不同，Mplus 可以：

  • 测试更复杂的模型结构，包括多个中介变量和/或多个调节变量的组合，以及更复杂的交互项。
  • 将这些模型扩展到多层数据结构。有关多层中介模型的示例和优秀论文，请访问 Kris Preacher 的网站。

我建议您在复制代码的同时，努力理解代码，这样您就能很快学会如何将代码调整以满足您的需求。

当然，这种潜力和灵活性的缺点是您需要掌握 Mplus 编程技能。这就是本教程的意义所在！实际上，我认为在 Mplus 中进行中介和调节模型测试对研究人员的长远发展是有利的，因为编程会迫使您正确理解方法，从而更容易撰写报告并发现模型中的概念错误。

虽然本网站将帮助您编写和理解模型代码，但提供的代码和指南是为具有一定 Mplus 基础知识的人员设计的。在使用这些代码之前，您需要掌握初级的 Mplus 技能，例如如何将数据读入 Mplus，如何编码和处理缺失数据，如何估计模型，如何指定不同的结果分布，BY、ON、WITH 和 XWITH 语句以及 @ 和 () 符号的用法，以及 MODEL CONSTRAINT: 如何用于检验参数的函数。

为此（广告时间！），我会不定期举办关于 Mplus 基础知识以及使用 Mplus 或 SPSS 测试中介、调节和调节中介模型的公开课程，也会提供上门培训。当然，还有一些关于 Mplus 的好书（我推荐 Christian Geiser 的《Data Analysis with Mplus》），以及其他一些课程提供商也提供类似的入门课程。另外，从 2016 年 7 月起，statmodel.com（Mplus 的程序员）将出售一本概述各种回归场景（包括中介模型）的 Mplus 编码的书籍。

无论如何，我希望您觉得这些代码有帮助。非常欢迎对代码错误的任何更正。祝您分析顺利！

1.2 模型选择

本教程提供了多种不同结构的中介、调节和调节中介模型代码示例。在开始分析之前，选择最适合您的研究问题的模型至关重要。以下是一些需要考虑的因素：

  • 您主要研究的是中介效应还是调节效应？
    • 中介模型关注的是一个变量如何通过另一个变量影响结果。
    • 调节模型关注的是一个变量如何改变另一个变量对结果的影响。
  • 模型中涉及多少个中介变量和调节变量？
    • 本教程提供了包含单个中介变量和多个中介变量的模型示例。
    • 同样，本教程也提供了包含单个调节变量和多个调节变量的模型示例。
  • 是否存在调节中介效应？
    • 调节中介效应指的是一个调节变量会影响中介效应的强度或方向。

在选择模型时，请仔细审查本教程中提供的每个模型的概念图、统计模型图和模型方程。确保您选择了能够最准确反映您的理论和研究假设的模型。

1.3 模型索引

本教程的代码示例按照 Andrew Hayes 的 PROCESS 宏中模型的编号系统进行组织。但是，请记住，Mplus 的灵活性允许您测试比 PROCESS 宏中提供的更多模型结构。

为了方便起见，以下列出了一些关键的模型类型，并简要描述了它们的基本结构：

  • 简单中介模型： 包含一个自变量（X）、一个中介变量（M）和一个因变量（Y）。X 通过 M 影响 Y。
  • 简单调节模型： 包含一个自变量（X）、一个调节变量（W）和一个因变量（Y）。W 调节 X 对 Y 的影响。
  • 调节中介模型： 包含一个自变量（X）、一个中介变量（M）、一个调节变量（W）和一个因变量（Y）。W 调节 X 对 M 的影响，或 M 对 Y 的影响，或两者都调节。
  • 包含多个中介变量的模型： 包含一个自变量（X）、多个中介变量（M1、M2…）和一个因变量（Y）。
  • 包含多个调节变量的模型： 包含一个自变量（X）、多个调节变量（W1、W2…）和一个因变量（Y）。

本教程中提供了这些模型以及其他更复杂配置的 Mplus 代码示例。请仔细阅读每个模型的详细说明，并选择最适合您研究的模型。

温馨提示：

  • 在开始使用本教程中的代码之前，请确保您具备 Mplus 的基本知识，包括数据读取、变量定义和基本模型构建。
  • 请务必仔细检查您复制和粘贴的代码，以避免任何细微错误。
  • 尝试理解代码，而不仅仅是盲目地复制粘贴。这将使您能够更好地应用这些技术，并根据您自己的研究需求进行调整。
