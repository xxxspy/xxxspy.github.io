大家好，欢迎来到本教程。本教程将带大家学习如何使用 Mplus 构建一个复杂的模型，它包含中介效应、调节效应以及调节中介效应。 我们将以一个包含潜变量的案例进行讲解，这个案例中，自变量、中介变量、调节变量和因变量都是由多个观察变量构成的潜变量。 本教程的结构非常清晰，首先我们会讲解这个模型的理论基础，也就是理论模型部分。然后，我们会用数学公式来表达这个模型，这就是数学模型部分，并对其中的公式进行详细的推导，让大家清楚地理解每个参数的含义和作用。 接着，我们会讲解 Mplus 代码的编写，以及代码中每一个语句的作用，这就是代码解读部分。通过本教程，即使你对 Mplus 不熟悉，也能快速上手并分析你的数据。 希望大家能学有所获！如有任何问题，请随时提出。

理论模型

我们来看这张图，它展示了我们今天要分析的模型。 这是一个包含潜变量的中介、调节以及调节中介模型。 X是自变量，Y是因变量，M是中介变量，W和Z是调节变量。箭头表示变量间的关系，例如，X会影响M，X也会直接影响Y，M也会影响Y。 W和Z分别调节X对M的影响，W还调节X对Y的直接影响以及M对Y的影响。 此外，Z还会直接影响M。所有这些影响都是正向的。

数学模型

这张图展示了我们研究的数学模型，它包含了潜变量的中介和调节效应。

首先，X代表我们的自变量，也就是我们想要研究影响因素。M是中介变量，它在X和Y之间起桥梁作用。Y是因变量，也就是我们想要预测的结果。

W和Z是两个调节变量，它们会影响X对M的影响以及M对Y的影响，甚至X对Y的直接影响。

XW、XZ和MW是交互项，它们分别代表X和W、X和Z以及M和W之间的交互作用。

图中的箭头表示变量之间的影响方向，箭头上的字母代表路径系数。例如，a1表示X对M的影响，b1表示M对Y的影响，c1’表示X对Y的直接影响。这个模型综合考虑了直接效应、间接效应（也就是中介效应）以及调节效应。

数学公式1

Y = b0 + b1M + b2MW + c1'X + c2'W + c3'XW

这是我们的第一个模型方程，描述Y如何受M、W、X的影响。其中M是中介变量，W和X是自变量，Y是因变量。b0, b1, b2, c1’, c2’, c3’ 都是待估参数。

数学公式2

Y = b0 + b1M + b2MW + c1'X + c2'W + c3'XW
M = a0 + a1X + a2W + a3Z + a4XW + a5XZ

这是第二个模型方程，描述中介变量M如何受X、W、Z的影响。 a0, a1, a2, a3, a4, a5 都是待估参数。Z是另一个调节变量。

数学公式3

Y = b0 + b1M + b2MW + c1'X + c2'W + c3'XW
M = a0 + a1X + a2W + a3Z + a4XW + a5XZ
Y = b0 + b1(a0 + a1X + a2W + a3Z + a4XW + a5XZ) + b2(a0 + a1X + a2W + a3Z + a4XW + a5XZ)W + c1'X + c2'W + c3'XW

将第二个方程代入第一个方程，开始进行代数推导。

数学公式4

Y = b0 + b1M + b2MW + c1'X + c2'W + c3'XW
M = a0 + a1X + a2W + a3Z + a4XW + a5XZ
Y = b0 + b1(a0 + a1X + a2W + a3Z + a4XW + a5XZ) + b2(a0 + a1X + a2W + a3Z + a4XW + a5XZ)W + c1'X + c2'W + c3'XW
Y = b0 + a0b1 + a1b1X + a2b1W + a3b1Z + a4b1XW + a5b1XZ + a0b2W + a1b2XW + a2b2WW + a3b2ZW + a4b2XWW + a5b2XZW + c1'X + c2'W + c3'XW

展开括号，得到一个更复杂的表达式。

数学公式5

Y = b0 + b1M + b2MW + c1'X + c2'W + c3'XW
M = a0 + a1X + a2W + a3Z + a4XW + a5XZ
Y = b0 + b1(a0 + a1X + a2W + a3Z + a4XW + a5XZ) + b2(a0 + a1X + a2W + a3Z + a4XW + a5XZ)W + c1'X + c2'W + c3'XW
Y = b0 + a0b1 + a1b1X + a2b1W + a3b1Z + a4b1XW + a5b1XZ + a0b2W + a1b2XW + a2b2WW + a3b2ZW + a4b2XWW + a5b2XZW + c1'X + c2'W + c3'XW
Y = (b0 + a0b1 + a2b1W + a3b1Z + a0b2W + a2b2WW + a3b2ZW + c2'W) + (a1b1 + a4b1W + a5b1Z + a1b2W + a4b2WW + a5b2ZW + c1' + c3'W)X

整理公式，使其符合Y = a + bX的形式，以便提取X对Y的影响。

数学公式6

Y = b0 + b1M + b2MW + c1'X + c2'W + c3'XW
M = a0 + a1X + a2W + a3Z + a4XW + a5XZ
Y = b0 + b1(a0 + a1X + a2W + a3Z + a4XW + a5XZ) + b2(a0 + a1X + a2W + a3Z + a4XW + a5XZ)W + c1'X + c2'W + c3'XW
Y = b0 + a0b1 + a1b1X + a2b1W + a3b1Z + a4b1XW + a5b1XZ + a0b2W + a1b2XW + a2b2WW + a3b2ZW + a4b2XWW + a5b2XZW + c1'X + c2'W + c3'XW
Y = (b0 + a0b1 + a2b1W + a3b1Z + a0b2W + a2b2WW + a3b2ZW + c2'W) + (a1b1 + a4b1W + a5b1Z + a1b2W + a4b2WW + a5b2ZW + c1' + c3'W)X
X对Y的间接效应（在W，Z的条件下）：(a1 + a4W + a5Z)(b1 + b2W)

间接效应代表X通过M对Y的影响。可以看到，它受W和Z的调节。

数学公式7

Y = b0 + b1M + b2MW + c1'X + c2'W + c3'XW
M = a0 + a1X + a2W + a3Z + a4XW + a5XZ
Y = b0 + b1(a0 + a1X + a2W + a3Z + a4XW + a5XZ) + b2(a0 + a1X + a2W + a3Z + a4XW + a5XZ)W + c1'X + c2'W + c3'XW
Y = b0 + a0b1 + a1b1X + a2b1W + a3b1Z + a4b1XW + a5b1XZ + a0b2W + a1b2XW + a2b2WW + a3b2ZW + a4b2XWW + a5b2XZW + c1'X + c2'W + c3'XW
Y = (b0 + a0b1 + a2b1W + a3b1Z + a0b2W + a2b2WW + a3b2ZW + c2'W) + (a1b1 + a4b1W + a5b1Z + a1b2W + a4b2WW + a5b2ZW + c1' + c3'W)X
X对Y的间接效应（在W，Z的条件下）：(a1 + a4W + a5Z)(b1 + b2W)
X对Y的直接效应（在W的条件下）：c1' + c3'W

直接效应代表X对Y的直接影响，同样受W的调节。

代码解读1

USEVARIABLES = X1 X2 X3 X4 M1 M2 M3 M4 W1 W2 W3 W4 Z1 Z2 Z3 Z4 Y1 Y2 Y3 Y4;

首先，我们用USEVARIABLES语句声明所有要用到的变量。包括自变量X（X1到X4），中介变量M（M1到M4），调节变量W和Z（分别用W1到W4和Z1到Z4表示），以及因变量Y（Y1到Y4）。

代码解读2

USEVARIABLES = X1 X2 X3 X4 M1 M2 M3 M4 W1 W2 W3 W4 Z1 Z2 Z3 Z4 Y1 Y2 Y3 Y4;
ANALYSIS:
TYPE = GENERAL RANDOM;
ESTIMATOR = ML;
ALGORITHM = INTEGRATION;

接下来是分析设置部分。我们选择’GENERAL RANDOM’类型，表示这是一个包含随机效应的模型；用最大似然估计法(‘ML’)进行参数估计；并采用数值积分法(‘INTEGRATION’)来处理潜在变量。

代码解读3

USEVARIABLES = X1 X2 X3 X4 M1 M2 M3 M4 W1 W2 W3 W4 Z1 Z2 Z3 Z4 Y1 Y2 Y3 Y4;
ANALYSIS:
TYPE = GENERAL RANDOM;
ESTIMATOR = ML;
ALGORITHM = INTEGRATION;
MODEL:
X BY X1 X2 X3 X4;
M BY M1 M2 M3 M4;
W BY W1* W2 W3 W4;
Z BY Z1* Z2 Z3 Z4;
Y BY Y1 Y2 Y3 Y4;

这里定义了测量模型，将潜变量X、M、W、Z和Y与它们的观测变量对应起来。注意，W1和Z1后面有个星号(*)，表示我们将W和Z的方差固定为1，也就是进行标准化处理。

代码解读4

USEVARIABLES = X1 X2 X3 X4 M1 M2 M3 M4 W1 W2 W3 W4 Z1 Z2 Z3 Z4 Y1 Y2 Y3 Y4;
ANALYSIS:
TYPE = GENERAL RANDOM;
ESTIMATOR = ML;
ALGORITHM = INTEGRATION;
MODEL:
X BY X1 X2 X3 X4;
M BY M1 M2 M3 M4;
W BY W1* W2 W3 W4;
Z BY Z1* Z2 Z3 Z4;
Y BY Y1 Y2 Y3 Y4;
W@1;  Z@1;

这一行代码与上一步骤中的星号(*)作用相同，再次明确固定W和Z的方差为1，确保它们被标准化。

代码解读5

USEVARIABLES = X1 X2 X3 X4 M1 M2 M3 M4 W1 W2 W3 W4 Z1 Z2 Z3 Z4 Y1 Y2 Y3 Y4;
ANALYSIS:
TYPE = GENERAL RANDOM;
ESTIMATOR = ML;
ALGORITHM = INTEGRATION;
MODEL:
X BY X1 X2 X3 X4;
M BY M1 M2 M3 M4;
W BY W1* W2 W3 W4;
Z BY Z1* Z2 Z3 Z4;
Y BY Y1 Y2 Y3 Y4;
W@1;  Z@1;
MW | M XWITH W;
XW | X XWITH W;
XZ | X XWITH Z;

现在我们定义潜变量的交互项。’XWITH’表示构建两个潜变量的交互作用。例如，’MW | M XWITH W’表示创建中介变量M和调节变量W的交互项。

代码解读6

USEVARIABLES = X1 X2 X3 X4 M1 M2 M3 M4 W1 W2 W3 W4 Z1 Z2 Z3 Z4 Y1 Y2 Y3 Y4;
ANALYSIS:
TYPE = GENERAL RANDOM;
ESTIMATOR = ML;
ALGORITHM = INTEGRATION;
MODEL:
X BY X1 X2 X3 X4;
M BY M1 M2 M3 M4;
W BY W1* W2 W3 W4;
Z BY Z1* Z2 Z3 Z4;
Y BY Y1 Y2 Y3 Y4;
W@1;  Z@1;
MW | M XWITH W;
XW | X XWITH W;
XZ | X XWITH Z;
Y ON M (b1); ... M ON XZ (a5);

这一部分定义了结构模型，描述了变量之间的关系以及路径系数。例如，’Y ON M (b1)’表示M对Y有直接影响，其路径系数命名为b1。其他语句以此类推。

代码解读7

USEVARIABLES = X1 X2 X3 X4 M1 M2 M3 M4 W1 W2 W3 W4 Z1 Z2 Z3 Z4 Y1 Y2 Y3 Y4;
ANALYSIS:
TYPE = GENERAL RANDOM;
ESTIMATOR = ML;
ALGORITHM = INTEGRATION;
MODEL:
X BY X1 X2 X3 X4;
M BY M1 M2 M3 M4;
W BY W1* W2 W3 W4;
Z BY Z1* Z2 Z3 Z4;
Y BY Y1 Y2 Y3 Y4;
W@1;  Z@1;
MW | M XWITH W;
XW | X XWITH W;
XZ | X XWITH Z;
Y ON M (b1); ... M ON XZ (a5);
MODEL CONSTRAINT:
NEW(LOW_W MED_W HIGH_W LOW_Z MED_Z HIGH_Z ...);
LOW_W = -1; ... HIGH_Z = 1; ...

接下来是模型约束部分。我们定义了调节变量W和Z的三个水平：低、中、高，分别用-1、0、1表示，这假设数据已经标准化。后续的语句会用到这些定义。

代码解读8

USEVARIABLES = X1 X2 X3 X4 M1 M2 M3 M4 W1 W2 W3 W4 Z1 Z2 Z3 Z4 Y1 Y2 Y3 Y4;
ANALYSIS:
TYPE = GENERAL RANDOM;
ESTIMATOR = ML;
ALGORITHM = INTEGRATION;
MODEL:
X BY X1 X2 X3 X4;
M BY M1 M2 M3 M4;
W BY W1* W2 W3 W4;
Z BY Z1* Z2 Z3 Z4;
Y BY Y1 Y2 Y3 Y4;
W@1;  Z@1;
MW | M XWITH W;
XW | X XWITH W;
XZ | X XWITH Z;
Y ON M (b1); ... M ON XZ (a5);
MODEL CONSTRAINT:
NEW(LOW_W MED_W HIGH_W LOW_Z MED_Z HIGH_Z ...);
LOW_W = -1; ... HIGH_Z = 1; ...
ILOW_LOZ = a1*b1 + a4*b1*LOW_W + ...; ... IHIW_HIZ = ...;

这里计算的是在不同调节变量组合下的条件间接效应。这些复杂的公式是根据模型推导出来的，用于计算间接效应在不同调节变量水平下的值。
