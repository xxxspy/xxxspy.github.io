大家好，欢迎来到Mplus非线性中介效应分析教程。本教程将深入讲解如何使用Mplus处理预测变量对中介变量和结果变量存在非线性影响的中介模型。

本教程的结构设计如下：首先，我们将介绍模型的理论基础，阐明我们研究的变量关系和背后的逻辑。接下来，我们会展示该模型的数学表达形式，让大家了解模型是如何用数学语言描述的。然后，我们将推导模型中的关键效应，如直接效应和间接效应，并特别关注如何处理预测变量的非线性影响。最后，我们会逐行解读Mplus代码，解释如何使用Mplus来实现模型分析，并进行结果的解读。

通过本教程，您将掌握使用Mplus分析包含非线性效应的中介模型的完整流程，并能够有效地解释和可视化模型结果。希望大家能从本教程中获益，更好地理解复杂的中介关系。

理论模型

我们来看，这里展示了一个模型，X变量会直接影响Y变量。同时，X也会影响M变量，然后M变量再影响Y变量。此外，X的平方项也会直接影响Y变量，也会影响M变量，最终M变量影响Y变量。箭头表示变量之间的影响方向。

数学模型

这张图展示了一个变量关系的数学模型。其中，X 代表一个预测变量，它通过两条路径影响中间变量 M：一条是X本身的影响，用a1表示；另一条是X的平方项X²的影响，用a2表示。同时，X还通过路径c1’直接影响结果变量Y。而中间变量M通过路径b1影响结果变量Y，并且X也直接通过c2’影响结果变量Y。这个模型主要研究X，M和Y之间复杂的关联，特别关注X对M和Y的非线性影响。

数学公式1

模型公式:
Y = b0 + b1M + c1'X + c2'XX
M = a0 + a1X + a2XX

首先，我们定义了两个回归模型。第一个模型描述了结果变量Y如何受中介变量M、预测变量X以及X的平方的影响。第二个模型则描述了中介变量M如何受预测变量X及其平方的影响。这里的b0, b1, a0, a1, a2, c1’, c2’ 都是回归系数。

数学公式2

模型公式:
Y = b0 + b1M + c1'X + c2'XX
M = a0 + a1X + a2XX
求导计算:
dY/dX = c1' + 2c2'X
dY/dM = b1
dM/dX = a1 + 2a2X

为了分析X如何影响Y，我们对模型中的方程进行求导。第一个偏导数dY/dX表示X对Y的直接影响；第二个偏导数dY/dM表示M对Y的影响；第三个偏导数dM/dX则表示X对M的影响。注意，这里我们使用了偏导数，因为Y和M都受多个变量的影响。

数学公式3

模型公式:
Y = b0 + b1M + c1'X + c2'XX
M = a0 + a1X + a2XX
求导计算:
dY/dX = c1' + 2c2'X
dY/dM = b1
dM/dX = a1 + 2a2X
计算间接效应:
瞬时间接效应 (IIE) = (a1 + 2a2X)b1

间接效应指的是X通过M影响Y的路径。我们通过将dM/dX和dY/dM相乘来计算瞬时间接效应，也就是X通过M间接影响Y的程度。这里可以看到间接效应的值会随着X的值而变化，因此称为瞬时间接效应。

数学公式4

模型公式:
Y = b0 + b1M + c1'X + c2'XX
M = a0 + a1X + a2XX
求导计算:
dY/dX = c1' + 2c2'X
dY/dM = b1
dM/dX = a1 + 2a2X
计算间接效应:
瞬时间接效应 (IIE) = (a1 + 2a2X)b1
计算直接效应:
瞬时直接效应 (IDE) = c1' + 2c2'X

直接效应指的是X直接影响Y的路径，也就是dY/dX。这里我们称之为瞬时直接效应，因为它也会随着X值的变化而变化。

代码解读1

! Predictor variable(s) - X, XX
! Mediator variable(s)  M
! Moderator variable(s) - none
! Outcome variable - Y

第一步，我们来看这段注释。它告诉我们模型中的变量：X是预测变量，XX是X的平方项，M是中介变量，Y是结果变量。这个模型中没有调节变量。

代码解读2

! Predictor variable(s) - X, XX
! Mediator variable(s)  M
! Moderator variable(s) - none
! Outcome variable - Y
USEVARIABLES = X XX M Y;

第二步，USEVARIABLES命令指定了分析中实际使用的变量，包括X、XX、M和Y。这些是我们接下来模型构建要用到的变量。

代码解读3

! Predictor variable(s) - X, XX
! Mediator variable(s)  M
! Moderator variable(s) - none
! Outcome variable - Y
USEVARIABLES = X XX M Y;
ANALYSIS:
 TYPE = GENERAL;
 ESTIMATOR = ML;
 BOOTSTRAP = 10000;

第三步，ANALYSIS代码块定义了分析的类型和估计方法。TYPE = GENERAL; 表示这是一个普通的模型分析。ESTIMATOR = ML; 指定使用最大似然法进行参数估计。BOOTSTRAP = 10000; 表示执行10000次自助抽样，用于计算参数估计的标准误差和置信区间。

代码解读4

! Predictor variable(s) - X, XX
! Mediator variable(s)  M
! Moderator variable(s) - none
! Outcome variable - Y
USEVARIABLES = X XX M Y;
ANALYSIS:
 TYPE = GENERAL;
 ESTIMATOR = ML;
 BOOTSTRAP = 10000;
MODEL:
 Y ON M (b1);
 Y ON X (cdash1); 
! direct effect of X on Y
 Y ON XX (cdash2); 
! direct effect of X on Y
 M ON X (a1);
 M ON XX (a2);

第四步，MODEL代码块定义了模型中变量之间的关系。Y ON M (b1); 表示Y对M的回归，回归系数标记为b1。Y ON X (cdash1); 表示Y对X的回归，系数为cdash1，代表X对Y的直接效应。Y ON XX (cdash2); 表示Y对X的平方项XX的回归，系数为cdash2。M ON X (a1); 表示M对X的回归，系数为a1。M ON XX (a2); 表示M对X的平方项XX的回归，系数为a2。

代码解读5

! Predictor variable(s) - X, XX
! Mediator variable(s)  M
! Moderator variable(s) - none
! Outcome variable - Y
USEVARIABLES = X XX M Y;
ANALYSIS:
 TYPE = GENERAL;
 ESTIMATOR = ML;
 BOOTSTRAP = 10000;
MODEL:
 Y ON M (b1);
 Y ON X (cdash1); 
! direct effect of X on Y
 Y ON XX (cdash2); 
! direct effect of X on Y
 M ON X (a1);
 M ON XX (a2);
MODEL CONSTRAINT:
 NEW(LOW_X MED_X HIGH_X
 IIE_LOWX IIE_MEDX IIE_HIX
 IDE_LOWX IDE_MEDX IDE_HIX);
 LOW_X = #LOWX;
! replace #LOWX in the code with your chosen low value of X
 MED_X = #MEDX;
! replace #MEDX in the code with your chosen medium value of X
 HIGH_X = #HIGHX;
! replace #HIGHX in the code with your chosen high value of X
! Calc instantaneous indirect effects for low, medium, high values of X
 IIE_LOWX = (a1 + 2*a2*LOW_X)*b1;
 IIE_MEDX = (a1 + 2*a2*MED_X)*b1;
 IIE_HIX = (a1 + 2*a2*HIGH_X)*b1;
! Calc instantaneous direct effects for low, medium, high values of X
 IDE_LOWX = cdash1 + 2*cdash2*LOW_X;
 IDE_MEDX = cdash1 + 2*cdash2*MED_X;
 IDE_HIX = cdash1 + 2*cdash2*HIGH_X;

第五步，MODEL CONSTRAINT 代码块定义新的参数，并计算不同X值下的即时间接效应和即时直接效应。NEW 定义了新参数，LOW_X，MED_X和HIGH_X分别代表X的低中高值。IIE系列代表即时间接效应，IDE系列代表即时直接效应。LOW_X，MED_X，HIGH_X的值需要用户替换代码中的#LOWX，#MEDX和#HIGHX。之后，代码根据公式计算了在不同X值下的即时间接效应和即时直接效应。

第六步，这段代码使用LOOP循环来计算一系列X值对应的即时间接效应(IIEX)，并用PLOT命令绘制。PLOT(IIEX);表示绘制名为IIEX的变量。LOOP(XVAL,1,5,0.1); 创建一个循环，XVAL从1迭代到5，步长为0.1。注意，1和5需要替换为分析中X的最小值和最大值。循环中，针对每个XVAL计算对应的即时间接效应IIEX。
