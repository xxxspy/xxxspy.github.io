使用Mplus进行简单调节效应分析

大家好，欢迎来到本期教程。今天我们将学习如何使用Mplus软件进行简单的调节效应分析，特别是包含潜变量的模型。这个教程由谢菲尔德大学工作心理学研究所的Figure it out统计咨询公司提供。

首先，我们将介绍模型的理论基础，也就是我们想研究的现象和变量之间的关系。然后，我们会建立相应的数学模型，用公式来表达这些关系。 接下来，我们将进行一些关键的数学推导，帮助大家理解模型是如何工作的，以及我们如何获得最终的结果。最后，我们将详细解读Mplus的代码，一步一步地讲解如何用Mplus软件来运行这个模型以及如何理解输出结果。 本教程假设大家已经具备基本的潜变量建模知识，我们会专注于调节效应分析的具体操作和解读，让大家能够快速上手。 希望大家通过本教程能够掌握使用Mplus构建和分析潜变量调节模型，并能准确理解其结果。

理论模型

我们来看这个模型图。它展示了一个简单的调节效应。X 变量直接影响 Y 变量。而 W 变量调节了 X 对 Y 的影响，也就是说，W 变量会改变X对Y影响的强度。需要注意的是，W 变量本身并不直接影响 Y 变量。

数学模型

这张图展示了我们的数学模型。 它包含三个预测变量：X，W，以及X和W的交互项XW，还有一个结果变量Y。 X对Y的影响会受到W的调节作用。 图中箭头表示变量间的影响方向，比如从X指向Y的箭头表示X会影响Y。 b1代表X对Y的直接效应；b2代表W对Y的直接效应；b3则代表X和W交互作用对Y的效应。 接下来，我们会更详细地解释这些参数的含义以及如何用Mplus进行计算和解读。

数学公式1

Y = b0 + b1X + b2W + b3XW

这是我们的初始模型方程，其中Y是因变量，X和W是自变量，b0, b1, b2, b3是回归系数。这个方程包含了X对Y的直接效应，W对Y的直接效应，以及X和W的交互效应。

数学公式2

Y = b0 + b1X + b2W + b3XW
Y = (b0 + b2W) + (b1 + b3W)X

为了计算间接效应和条件效应，我们将模型改写成Y = a + bX的形式。通过重新整理项，我们将方程改写成这个形式。其中 (b0 + b2W) 是新的截距项，(b1 + b3W) 是新的斜率项。

数学公式3

Y = b0 + b1X + b2W + b3XW
Y = (b0 + b2W) + (b1 + b3W)X
b1 + b3W

这就是在给定W的条件下，X对Y的直接效应。 请注意，这个直接效应不再是一个常数，而是依赖于W的值。 这就体现了W对X与Y之间关系的调节作用。

代码解读1

USEVARIABLES = X1 X2 X3 X4 W1 W2 W3 W4 Y1 Y2 Y3 Y4;

首先，我们定义分析中使用的变量。X1到X4测量潜在变量X，W1到W4测量潜在变量W，Y1到Y4测量潜在变量Y。这一行代码告诉Mplus我们要分析哪些变量。

代码解读2

USEVARIABLES = X1 X2 X3 X4 W1 W2 W3 W4 Y1 Y2 Y3 Y4;
ANALYSIS:

接下来，我们进入分析设置部分。ANALYSIS这个关键词标志着分析参数的开始。

代码解读3

USEVARIABLES = X1 X2 X3 X4 W1 W2 W3 W4 Y1 Y2 Y3 Y4;
ANALYSIS:
TYPE = GENERAL RANDOM;

这里指定模型类型为一般随机效应模型，这适合处理具有随机效应的数据。

代码解读4

USEVARIABLES = X1 X2 X3 X4 W1 W2 W3 W4 Y1 Y2 Y3 Y4;
ANALYSIS:
TYPE = GENERAL RANDOM;
ESTIMATOR = ML;

我们选择最大似然估计法 (ML) 来估计模型参数。这是结构方程模型常用的估计方法。

代码解读5

USEVARIABLES = X1 X2 X3 X4 W1 W2 W3 W4 Y1 Y2 Y3 Y4;
ANALYSIS:
TYPE = GENERAL RANDOM;
ESTIMATOR = ML;
ALGORITHM = INTEGRATION;

由于我们处理的是潜在变量模型，需要使用积分算法进行估计，以获得更准确的结果。

代码解读6

USEVARIABLES = X1 X2 X3 X4 W1 W2 W3 W4 Y1 Y2 Y3 Y4;
ANALYSIS:
TYPE = GENERAL RANDOM;
ESTIMATOR = ML;
ALGORITHM = INTEGRATION;
MODEL:

现在我们开始定义模型。MODEL关键词表示模型规范的开始。

代码解读7

USEVARIABLES = X1 X2 X3 X4 W1 W2 W3 W4 Y1 Y2 Y3 Y4;
ANALYSIS:
TYPE = GENERAL RANDOM;
ESTIMATOR = ML;
ALGORITHM = INTEGRATION;
MODEL:
X BY X1 X2 X3 X4;

这一行定义了潜在变量X的测量模型，它由观察变量X1到X4测量。

代码解读8

USEVARIABLES = X1 X2 X3 X4 W1 W2 W3 W4 Y1 Y2 Y3 Y4;
ANALYSIS:
TYPE = GENERAL RANDOM;
ESTIMATOR = ML;
ALGORITHM = INTEGRATION;
MODEL:
X BY X1 X2 X3 X4;
W BY W1* W2 W3 W4;

同样，我们定义潜在变量W的测量模型，由W1到W4测量。注意W1后面有个星号*，表示W1的载荷固定为1，用于标准化W。

代码解读9

USEVARIABLES = X1 X2 X3 X4 W1 W2 W3 W4 Y1 Y2 Y3 Y4;
ANALYSIS:
TYPE = GENERAL RANDOM;
ESTIMATOR = ML;
ALGORITHM = INTEGRATION;
MODEL:
X BY X1 X2 X3 X4;
W BY W1* W2 W3 W4;
Y BY Y1 Y2 Y3 Y4;

最后，我们定义潜在变量Y的测量模型，由Y1到Y4测量。

代码解读10

USEVARIABLES = X1 X2 X3 X4 W1 W2 W3 W4 Y1 Y2 Y3 Y4;
ANALYSIS:
TYPE = GENERAL RANDOM;
ESTIMATOR = ML;
ALGORITHM = INTEGRATION;
MODEL:
X BY X1 X2 X3 X4;
W BY W1* W2 W3 W4;
Y BY Y1 Y2 Y3 Y4;
W@1;

将W的方差固定为1，进一步标准化W。

代码解读11

USEVARIABLES = X1 X2 X3 X4 W1 W2 W3 W4 Y1 Y2 Y3 Y4;
ANALYSIS:
TYPE = GENERAL RANDOM;
ESTIMATOR = ML;
ALGORITHM = INTEGRATION;
MODEL:
X BY X1 X2 X3 X4;
W BY W1* W2 W3 W4;
Y BY Y1 Y2 Y3 Y4;
W@1;
XW | X XWITH W;

这里创建了潜在变量X和W的交互作用项XW，这是分析调节效应的关键。

代码解读12

W@1;
XW | X XWITH W;
Y ON X (b1);

指定X对Y的回归，参数命名为b1。

指定XW对Y的回归，参数命名为b3。这个参数反映了调节效应。

NEW(LOW_W MED_W HIGH_W SIMP_LO SIMP_MED SIMP_HI);
LOW_W = -1; MED_W = 0; HIGH_W = 1;

设置W的三个值：-1, 0, 1，代表W的低、中、高三个水平，用于计算简单斜率。

代码解读18

USEVARIABLES = X1 X2 X3 X4 W1 W2 W3 W4 Y1 Y2 Y3 Y4;
